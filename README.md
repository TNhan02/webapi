An ongoing backend side of a fullstack website. The frontend side can be found here: https://github.com/TNhan02/React

The website simulates an online shop selling products, which has 2 sides, admin and user. It uses React for the frontend and .NET Core 6.0 for the backend.
A database is created in Microsoft SQL Server. Each user role is allowed to fetch assigned data fields according to the role-based authorization.
